# Stickers Assets Directory

This directory contains sticker assets for the messaging application.

## Structure:
- emotions/ - Emotion stickers (happy, love, laugh, sad, angry)
- reactions/ - Reaction stickers (thumbs up, thumbs down, clap, fire, 100)
- animals/ - Animal stickers (dog, cat, monkey, lion, elephant)
- celebrations/ - Celebration stickers (party, confetti, birthday, gift)

## Usage:
Stickers are referenced in the code using paths like:
- 'assets/stickers/emotions/happy.png'
- 'assets/stickers/reactions/thumbs_up.png'
- etc.

Note: You'll need to add actual PNG image files to these directories and update your pubspec.yaml to include the assets.
